# MonkeyTest Report Reviewer Template

Use this structure when dispatching `monkey-test-report-reviewer`:

- Route: `{ROUTE}`
- Route slug: `{ROUTE_SLUG}`
- Report file: `{REPORT_FILE}`
- Screenshots dir: `{SCREENSHOTS_DIR}`
- Bug report output: `{BUG_REPORT_OUTPUT}`
- Slice index: `{SLICE_INDEX}`
- Total slices: `{TOTAL_SLICES}`
- Screenshot files: `{SCREENSHOT_FILES}`

Requirements:
- review exactly the assigned route or slice
- no browser usage
- inspect screenshots and route report together
- return a Markdown bug report
